PlaceMate is a web-based information portal designed to display placement-related information for Information Science and Engineering (ISE) students at Basaveshwar Engineering College, Bagalkot.

The project was developed as part of hands-on web development practice using HTML, CSS, and JavaScript, with Google Sheets used for basic data storage.

## Features
- Centralized placement information display
- Clean and user-friendly interface
- JavaScript-based interactivity
- Data management using Google Sheets
- Organized project structure with images folder

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Sheets (for data storage)

## How to Run
1. Download or clone the repository
2. Open `index.html` in a web browser
